primeiro inicie o venv

.venv/Scripts/activate

apos isso baixe essas bibliotecas

pip install flask flask-login mysql-connector

apos isso abra o prompt e digite "flask run" obviamente so o comando sem as aspas

se nao for tente inicializar pelo app.py

e se nao for tambem digite no promp "python -m flask --app sitedelivros run" obviamente so o comando sem as aspas
